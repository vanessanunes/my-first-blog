# my-first-blog

Feito em DJango

**Instruções**


## Iniciar ambiente de desenvolvimento
> . myvenv/bin/activate


## Instalar requirements.txt
> pip3 install requirements.txt

## Iniciar aplicação no servidor local
> python3 manage.py runserver
